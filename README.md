# Rusty-Raft
An implemention of the [Raft consensus algorithim](https://raft.github.io) in rust. 

## Building
`cargo build`

## Testing
`cargo test`

## Module Structure
```
rusty_raft
  |-- rpc
  |-- client
  |-- server
```
